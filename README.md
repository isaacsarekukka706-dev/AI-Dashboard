# AI Dashboard - Student Performance Analysis

## Project Overview

This project is an interactive AI Dashboard built using Streamlit and Plotly. It analyzes student performance data and provides meaningful insights through KPIs, visualizations, and interactive filters. The dashboard helps users understand student performance based on marks, attendance, and assignment completion.

## Features

* Dataset Overview
* Data Cleaning (Null Values & Duplicate Removal)
* Key Performance Indicators (KPIs)
* Interactive Filters
* Multiple Data Visualizations
* User-Friendly Dashboard Interface

## Technologies Used

* Python
* Pandas
* NumPy
* Plotly
* Matplotlib
* Streamlit

## Dataset

The dataset contains student information such as:

* Student ID
* Gender
* Class
* Marks
* Attendance
* Assignments Completed

## Dashboard Components

### Dataset Overview

* Total Students
* Average Marks
* Average Attendance
* Average Assignments

### KPIs

* Highest Marks
* Lowest Marks
* Average Marks
* Pass Percentage

### Visualizations

1. Marks Distribution Histogram
2. Attendance vs Marks Scatter Plot
3. Assignment Analysis Bar Chart
4. Attendance Group Comparison Chart
5. Correlation Heatmap

### Interactive Filters

* Gender Filter
* Class Filter
* Marks Filter
* Attendance Filter

## How to Run

### 1. Clone the Repository

```bash
git clone <repository-link>
```

### 2. Install Required Libraries

```bash
pip install -r requirements.txt
```

### 3. Run the Application

```bash
streamlit run app.py
```
